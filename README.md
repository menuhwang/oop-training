# OOP 연습

## 주자창 시스템 구현

### 객체
- Parking: 주차장
    > cars: 현재 주차된 차량
- ChargePolicy: 요금 정잭
  > basicTime: 기본 시간
  > 
  > basicCharge: 기본 요금
  > 
  > unitTime: 추가 시간 단위
  > 
  > unitCharge: 추가 시간 당 요금
- Car: 차
    > id: 차량 번호
- Time: 입출차 시간
  > hour: 시
  >
  > minute: 분

### 기능

#### 입차

- 입차 로그

#### 출차

- 출차 로그

#### 요금 계산

유료 주차장으로 전환!

- 기본 이용시간
- 기본 요금
- 추가 시간 단위
- 추가 시간 당 요금
